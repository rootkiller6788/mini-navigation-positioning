# Course Tree - mini-inertial-navigation

## Prerequisites

### Mathematics
- Linear algebra: vector/matrix ops, eigenvalues
- Calculus: ODE, Taylor series, numerical integration
- Statistics: random processes, estimation, least-squares
- Lie theory: SO(3) rotation group, quaternion double cover

### Physics
- Newtonian mechanics: force, acceleration, rotating frames
- Geodesy: WGS84 ellipsoid, gravity field
- Sensor physics: MEMS accelerometer/gyroscope principles

### Signal Processing
- Stochastic processes: white noise, random walk, Allan variance
- Kalman filtering: linear estimation, covariance propagation
- Complementary filtering: frequency-domain sensor fusion
